Auto email teachers

A system to Keep on top of your work whilst you're away from school.

How is this innovative?

There was a need for this software to be developed because there is a pandemic currently and people are missing school for a week because of it. And as a result, miss important school work and fall behind. This is where our software come into play, our software allows users to imput their classes into a CSV file, with their teachers name and class that they missed, and then the script will read the CSV file and substitute the values which are stored in the File to an email, with other text and send it to all of the teachers with one click of a button.

Microsoft exchange servers.

This python script Connects to Microsoft’s exchange servers, which is what the school uses for their email service. It connects securely to the servers the same way you or I would in outlook! It uses the same user credentials to log in and send emails on your behalf

Bulk emails

This solution saves you from having to type out 5 different emails to 5 different people, you can send one bulk general email requesting the work that you’ve missed out on. This is especially helpful when there is a pandemic going on, or a slightly off-nominal flu season.
