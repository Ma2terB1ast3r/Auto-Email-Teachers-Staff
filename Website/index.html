Auto email teachers

This all started When Charlie and Thomas were too lazy to email their teachers when they were sick and away from school, so they decided to make a way to do this automatically.

How is this innovative?

There was a need for this software to be developed because there is a pandemic currently and people are missing school for a week because of it. And as a result, miss important school work and fall behind. This is where our software come into play, our software allows users to imput their classes into a CSV file, with their teachers name and class that they missed, and then the script will read the CSV file and substitute the values which are stored in the File to an email, with other text and send it to all of the teachers with one click of a button.
