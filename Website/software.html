How to install

Download the Repository

Download the files from: https://github.com/Ma2terB1ast3r/Auto-Email-Teachers-Staff

 

Extract the files

Right click on the zip file and extract the files into a folder.

Install python

Download Python from the official website (instructions are included on the website): https://www.python.org/downloads

Install required Libraries

Run the command to install the libraries whilst in the file directory (inside the folder where you extracted the files).

pip install -r requirements.txt

Run the .PY file

Double click on the file and it should open, to get all the details into the UI you will need to fill in the procvided CSV files and then you should be good.
