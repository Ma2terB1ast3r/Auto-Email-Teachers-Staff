How to install

Download the Repository

https://github.com/Ma2terB1ast3r/Auto-Email-Teachers-Staff

 

Extract the files

Right click on the files and select Extract here

Install python

https://www.python.org/downloads

Install required Libraries

Run the command to install the libraries whilst in the correct directory (wherever you have put the file)

pip install -r requirements.txt

Run the .PY file

Double click on the file and it should open, to get all the details into the UI you will need to fill in the procvided CSV files and then you should be good.
